Gravity Simulator, writtern with p5.js, TypeScript and Vite.

## Running it locally
### Prerequisites
- [Node.js](https://nodejs.org/en)
- pnpm ([Install](https://pnpm.io/installation))

### Steps
0. `pnpm i`
1. `pnpm run dev`
2. Open the url given by vite, usually http://localhost:8080/
